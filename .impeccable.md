## Design Context

### Users
- ผู้หญิงวัยทำงาน (working-age women) ที่ซื้อเสื้อผ้าแฟชั่นออนไลน์
- ใช้งานทั้ง desktop และ mobile ในเวลาว่างหรือพักเบรก
- ต้องการประสบการณ์ช้อปปิ้งที่รู้สึกพรีเมียมแต่ไม่ซับซ้อน

### Brand Personality
- **Minimal** — สะอาด เรียบ ไม่รก ทุก element มีเหตุผล
- ให้ความรู้สึก editorial fashion magazine มากกว่า e-commerce ทั่วไป
- ขาว-ดำเป็นแกนหลัก photography เป็นตัวให้สีสัน

### Aesthetic Direction
- **Reference**: Reformation — editorial, typography-driven, generous whitespace
- **Theme**: Light mode only
- **Color palette**: ขาว-ดำเป็นหลัก (เปลี่ยนจาก brown #3C2415)
- **Typography**: Agatho Light CAPS (display/serif) + PK Maehongson (body/Thai) — รักษาไว้
- **Layout**: Grid ที่มีจังหวะ สลับ product grid กับ statement typography sections
- **Photography**: รูปสินค้าเป็นตัวให้สีสันและ contrast กับ layout ขาว-ดำ

### Design Principles
1. **Typography is the design** — ตัวหนังสือคือ element หลัก ใช้ขนาด, weight, tracking สร้าง hierarchy
2. **Whitespace is luxury** — พื้นที่ว่างคือความหรูหรา ไม่ยัดทุกอย่างลงไป
3. **Black and white backbone** — ขาว-ดำเป็นโครงสร้าง ให้ photography เป็นตัวเล่าเรื่อง
4. **Less is more** — ทุก element ต้องมีเหตุผล ถ้าเอาออกได้โดยไม่เสียอะไร ให้เอาออก
5. **Editorial over commercial** — ให้รู้สึกเหมือนอ่าน magazine ไม่ใช่ scroll Shopee
